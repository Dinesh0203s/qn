UNIT - III: VECTOR SPACES
Short Answer Questions (2-3 Marks)
1. Let V = R³, show that W = {(a₁, a₂, a₃): a₁ = a₃ + 2} is not a subspace of V.
2. Define vector space.
3. Define direct sum of W₁ and W₂.
4. Show that if x, y and z are vectors in a vector space V such that x + z = y + z then x = y.
5. Define basis and dimension of a vector space.
6. Let V = R⁴. Show that the vectors v₁ = (1,0,1,0), v₂ = (0,1,0,1), v₃ = (0,0,0,2) are linearly independent.
Long Answer Questions (10-15 Marks)
1. Let V be a vector space and W a subset of V. Identify W is a subspace of V if and only if the following 3 conditions hold: (a) 0 ∈ W, (b) x + y ∈ W whenever x ∈ W and y ∈ W, (c) cx ∈ W whenever c ∈ F, x ∈ W.
2. Let W₁ and W₂ be the subspaces of V. Identify W₁ ∪ W₂ is a subspace of V if and only if W₁ ⊆ W₂ or W₂ ⊆ W₁.
3. Identify the set W = {(a₁, a₂, a₃) ∈ R³: 2a₁ - 7a₂ + a₃ = 0} is a subspace of V.
4. Let W₁ and W₂ be subspaces of a finite-dimensional vector space V. Construct necessary and sufficient conditions on W₁ and W₂ so that dim(W₁ ∩ W₂) = dim(W₁).
5. Construct the following set is basis for P₂(R): {-1 + 2x + 4x², 3 - 4x - 10x², -2 - 5x - 6x²}.
6. Let V be the set of sequences {aₙ} of real numbers. For {aₙ}, {bₙ} ∈ V and any real t, define {aₙ} + {bₙ} = {aₙ + bₙ} and t{aₙ} = {taₙ}. Identify with these operations, V is a vector space over R.
7. Identify the following set is basis for P₂(R): {-1 - x + 2x², 2 + x - 2x², 1 - 2x + 4x²}.
8. Build replacement theorem on vector spaces.
9. Construct the set of all polynomials over a field F is a vector space V.
10. Let W₁ and W₂ be subspaces of a vector space V. Construct W₁ + W₂ is a subspace of V that contains both W₁ and W₂.
UNIT - IV: LINEAR TRANSFORMATION
Short Answer Questions (2-3 Marks)
1. Define Linear transformation.
2. Find the following are linear transformation T: R → R by T(x) = x².
3. Find the linear transformation determined by the following matrix T: V₂(R) → V₂(R) given by [cos θ -sin θ; sin θ cos θ] with respect to the standard basis.
Long Answer Questions (10-15 Marks)
1. Let V and W be the vector spaces and T: V → W be linear. Then show that N(T) and R(T) are subspaces of V and W respectively.
2. Illustrate T: V₂(R) → V₂(R) defined by T(a,b) = (2a - 3b, a + 4b) is a linear transformation.
3. Let T: V₃ → V₃(R) be a linear map defined by T(x₁, x₂, x₃) = (x₁ - x₂, 2x₂ + x₃, 0). Interpret N(T) and R(T). Also illustrate the nullity T and Rank T.
4. Let T: R² → R³ defined by T(a₁, a₂) = (a₁ + a₂, 0, 2a₁ - a₂). Show that T is linear.
5. Let T be the linear operator on V₃(R) defined by T[a₁; a₂; a₃] = [4a₁ + a₃; 2a₁ + 3a₂ + 2a₃; a₁ + 4a₃] and β be the standard basis for V₃(R). Interpret the Eigen space of T, corresponds to each Eigen value.
6. Explain Dimension theorem.
7. Let T: P₂(R) → P₃(R) be the linear transformation defined by T[f(x)] = 2f'(x) + 3∫₀ˣ f(x)dx. Show that T is one to one or not.
8. Illustrate the matrix P that Diagonalise the matrix A = [2 2 -7; 2 1 2; 0 1 3] by similarity transformation and hence find A⁴.
9. Illustrate the matrix P that Diagonalises the matrix A = [2 2 1; 1 3 1; 1 2 2] by similarity transformation and hence find A⁴.
10. Illustrate the matrix P that Diagonalise the matrix A = [2 1 -1; 1 1 -2; -1 -2 1] by means of an orthogonal transformation.
UNIT - V: INNER PRODUCT SPACES
Short Answer Questions (2-3 Marks)
1. Define Norm.
2. Define Inner product space.
3. Let V = R² and S = {(1,0), (0,1)}. Find S is orthonormal basis or not.
4. Write parallelogram law formula.
5. Define orthogonality.
Long Answer Questions (10-15 Marks)
1. Illustrate the matrix P that Diagonalises the matrix A = [3 -1 0; -1 2 -1; 0 -1 3] by means of an orthogonal transformation.
2. Illustrate Gram-Schmidt process to construct an orthonormal basis for V₃(R) with the standard inner product for the basis {V₁, V₂, V₃} where V₁ = (1,0,1), V₂ = (1,3,1), V₃ = (3,2,1).
3. Illustrate Gram-Schmidt process to construct an orthonormal basis for V₃(R) with the standard inner product for the basis {V₁, V₂, V₃} where V₁ = (2,1,0), V₂ = (4,1,0), V₃ = (4,0,1).
4. Let V = C³, with inner product ⟨x,y⟩ = x₁ȳ₁ + x₂ȳ₂ + x₃ȳ₃ where x = (x₁,x₂,x₃) & y = (y₁,y₂,y₃). Let x = (2,1+i,i) & y = (2-i,2,1+2i). Show:
(i) ⟨x,y⟩
(ii) ‖x‖ & ‖y‖
(iii) ‖x‖ + ‖y‖
(iv) Cauchy-Schwartz inequality & Triangle inequality.
5. Let V be an inner product space, for x,y,z ∈ V and c ∈ F, Show that whether the following are true:
(i) ⟨x, y + z⟩ = ⟨x,y⟩ + ⟨x,z⟩
(ii) ⟨x, cy⟩ = c⟨x,y⟩
(iii) ⟨x,0⟩ = ⟨0,x⟩ = 0
(iv) ⟨x,x⟩ = 0 if and only if x = 0
(v) ⟨x,y⟩ = ⟨x,z⟩ for all x ∈ V then y = z
